Peak Height

  • Go to Special Parameters. Click on Specific Dimensions. Select Max Region Length (Peak Height) .

  • Draw a Rectangle on the image where you want to measure Max Region Length.

  • Set threshold and object colour.

  • If you are not getting the required length, adjust Threshold and try to get the length properly where needed.

  • Right click on window and drag the line having length (Peak Height).

  • Enter valid data in Property Window and save the operation.
