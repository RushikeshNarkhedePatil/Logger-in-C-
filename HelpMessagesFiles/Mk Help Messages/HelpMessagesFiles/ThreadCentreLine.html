ThreadCentreLine

  • Click on Manual Measurements.

  • Click on Intersection Point in Property window.

  • Select ThreadCentreLine.

  • Set the value of Radius.






  • Draw a rectangle on diagram where thread is present. While applying thread parameter note that you should select atleast more than two threads from image and the arrow where you are drawing a rectangle should be directed upward or downward only.








  • Likely, you will get the ThreadCentreLine. Click on Ok button to save the parameter.
