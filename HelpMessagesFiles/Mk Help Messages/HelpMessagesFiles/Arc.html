Arc


  • Go to Manual Measurements.

  • Select Arc Parameter.

  • Draw a Rectangle where user wants to draw an Arc.

  • Note that while drawing an Arc the rectangle arrow can be upward or downward based on where you want to draw the Arc region.

  • Once you draw the rectangle, right click on the image and you will get the arc. You can adjsut the arc sector as per requirment.

  • If you are not getting Arc, Adjust the properties from input parameters.

  • By double clicking on it, you will get another arc in same sector so you can apply it properly.
