Laser Profiler Measurement

  • Once you get the Lazer Profiling image, consider Base, Depth and Top for measuring Height and Depth.

  • Click on Lazer Profiler Measurement. Go to LazerProfilingMeasurement. Click on LazerProfilingMeasurement.

  • By right clicking, draw a rectangle, zoom in and zoom out wisely for choosing points generated by Lazer Profiling. Draw rectangles for Base, Depth and Top respectively. Note that, while drawing rectangle, the rectangle should be streached from both side identically. This will detect and measure lines more accurately.

  • Thus, you will get the results as a Base, Depth and Top blue lines.

  • To find height, go to Line to Line Parameter (Refer from below) and select Base Line and Top Line respectively. Enter all valid data and Save it.

    • Refer Line To Line Parameter.

  • To find Depth, go to Line to Line Parameter (Refer from Above) and select Top Line and Depth Line respectively. Enter all valid data and Save it.
