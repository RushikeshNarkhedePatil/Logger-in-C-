Angle Resting Reference Line



  • For Angle Resting Reference Line, initially you need to draw an Axis Line With help of Lines. Along with axis line you need to draw lines which can form an Angle.

  • Refer Line Parameter.

  • Refer Axis Line Parameter.

  • After drawing axis Line and Lines which can form an Angle, choose where you want an Angle Resting Reference Line.

  • Go to Manual Measurement and select Angle Resting Reference Line

  • Serially, click two angle forming lines one after another and then Axis Line.

  • Output line will be Angle Resting Reference Line.If you are not getting the output line, adjust distance from input parameter.

  • Output line will be Angle Resting Reference Line.

  • Once you right click on Image, you will get the output line as a Angle Resting Reference Line.
